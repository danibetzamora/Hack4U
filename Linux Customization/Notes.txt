Instalación Bspwm y Sxhkd

	Primer comando de instalación de paquetes:

		apt install build-essential git vim xcb libxcb-util0-dev libxcb-ewmh-dev libxcb-randr0-dev libxcb-		icccm4-dev libxcb-keysyms1-dev libxcb-xinerama0-dev libasound2-dev libxcb-xtest0-dev libxcb-shape0-		dev

	Clonamos en "Descargas" del usuario no privilegiado los siguientes repositorios:

		git clone https://github.com/baskerville/bspwm.git

		git clone https://github.com/baskerville/sxhkd.git

	Entrar a cada uno de los directorios y hacer un "make" y un "sudo make install"

	Terminar con un sudo apt install bspwm


Configuración del Sxhkd

	Crear varias carpetas en el directorio ~/.config:

		mkdir ~/.config/{bspwm,sxhkd}
