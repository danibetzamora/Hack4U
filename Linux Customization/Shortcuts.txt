Shortcuts Bspwm y Sxhkd

	windows + d -> Ejecuta rofi (buscador de programas)

	windows + shift + f -> Abrir Firefox

	windows + enter -> Abre consola

	windows + q -> Cierra consola (Por defecto es windows + w)

	windows + flechas -> Cambia entre terminales 

	windows + números -> Cambia entre espacios de trabajo

	windows + shift + números -> Mueve la ventana seleccionada al espacio de trabajo indicado en el número

	windows + alt + flechas -> Cambia tamaño de la terminal

	windows + s -> Ventana flotante

		mantener windows + click izquierdo -> Mover ventana flotante

		mantener windows + click derecho -> Resize de la ventana flotante

	windows + t -> Devolver la terminal a tamaño normal
	
	windows + ctrl + alt + flechas -> Preselectores

	windows + ctrl + alt + espacio -> Cancelar preselector

	windows + ctrl + numeros -> Preselectores de distintos tamaños

	windows + escape -> Recarga los ficheros de configuración de sxhkd (cuando se modifican comandos)

	windows + shift + {q,r} -> Quita o Reinicia Bspwm

	windows + shift + g -> Hacer captura de pantalla

	windows + shift + x -> Bloquea la pantalla (con i3lock)


Shortcuts Kitty

	kitty +kitten icat fondo.png -> Mostrar imagen en terminal

	ctrl + shift + enter -> Abrir una consola rápida en la propia terminal
	
	ctrl + shift + w -> Cerrar la consola rápida de la propia terminal (o escribir exit)
	
	ctrl + shift + z -> Hace zoom a una consola que está en una terminal (si se vuelve a hacer quita el zoom)

	ctrl + shift + t -> Formato Powerline (Crea sesiones como tmux en una terminal)

		ctrl + shift + flechas -> Cambiar entre sesiones (también se puede hacer pinchando en ellas)
		
		ctrl + shift + alt + t -> Renombrar sesión

	ctrl + shift + r -> Cambiar tamaño de la consola abierta en la terminal de kitty

		W -> Wider
	
		N -> Narrowe

		T -> Taller

		S -> Shorter

		R -> Reset

	ctrl + shift + u -> Abre una consola de hexadecimales (se pega un hexadecimal, se le da a enter, y sale el icono)


Funciones Personalizadas zshrc (Polybar)

	settarget Mantis 10.10.1.15 -> Declara objetivo indicando nombre de la máquina e IP

	cleartarget -> Borra el objetivo del módulo de la polybar


Alias Definidos

	cat -> Es un bat, con cuadrículas y paginación

	catn -> Es el bat sin cuadrículas pero con paginación

	catnp -> Es el bat como un cat normal (sin cuadrículas ni paginación)

	ctrl + alt + selección con ratón -> Selecciona respetando la columna en la que se está


### Notas Extra ###

locate fichero -> Da todas las rutas del sistema que contengan dicho nombre

kill -9 -1 -> Mata los procesos y lleva a la pantalla de iniciar sesión

Escribir cat + ctrl + t -> Puedes escribir el nombre de un fichero y te salen todas las opciones del sistema para escoger

nvim -> Nuevo editor de texto

ranger -> Programa que permite visualizar de forma gráfica los directorios

i3lock-fancy -> Bloqueador de pantalla

escibir cualquier comando + doble esc -> Añade sudo al inicio del comando (plugin)
